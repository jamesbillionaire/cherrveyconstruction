# cherrveyconstruction

Static company website built with Next.js, TypeScript, Tailwind CSS, shadcn/ui, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to preview the site.

## Static Build

```bash
npm run build
```

The static export is generated in `out/` for cPanel deployment.

## Deployment

Update `.cpanel.yml` with your hosting paths, then build and copy the contents of `out/` to your web root.
